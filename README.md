# 2ring-markdown-extension README

The 2Ring Markdown Extension should be used for editing internal documentation files.

## Features

The extension contains support for the following markup:

```
Containers
::: attention
:::

::: notice
:::

Font Awesome icons
:fa-icon:
:fas-icon:
:far-icon:
:fab-icon:

[ ] for checkboxes
[x] for filled checkboxes

^button^ for buttons
```
